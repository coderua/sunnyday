# Sunny Day - 5 day weather forecast

## Demo
https://coderua.github.io/sunnyday/

## Project setup
```
npm install -g @vue/cli # A runtime dependency
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run unit tests
```
npm run test:unit
```
